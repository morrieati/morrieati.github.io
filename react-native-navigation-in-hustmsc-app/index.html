俱乐部APP的页面跳转——React Native的Navigation

发表于 2016-11-03 | |

前面把一些准备工作已经做好了，然后开始写页面路由和导航啦。不过写之前还是要纠结一下选哪个框架才好。可是 React Native 的可供选择的导航框架这么多，真是选择困难症哇😂！

Navigation框架选择

本来已经选定了 NavigationExperimental 了，但是着实有些复杂，而且资料不是很丰富，Facebook 自己那个 Demo 还是用老的语法写的，貌似是一年前更新的？所以我又从网上找了些资料，最后发现了一个不错的解决方案—— react-native-router-flux
