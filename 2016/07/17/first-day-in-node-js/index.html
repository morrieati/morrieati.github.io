npm 是 Node.js 的包管理工具，可以安装海量 node_modules 😄，比如我们接下来马上就要用到的一个用来方便搭建服务器的这个模块——

express-generator

用官方的话说，Express 是一个基于 Node.js 平台，快速、开放、极简的 web 开发框架。一般我们都采用这个框架来搭建我们的 Node.js 应用（网站）。而 express-generator 则是用来快速搭建起 Express 框架的一个 Node.js 模块。

-g 是 npm 的一个选项，表示以全局方式安装这个模块，而不仅仅是安装在某个 Project 里。因为有些模块的作用是可以“超越”某个特定项目的，或者说，是很通用的，所以一般把它以全局方式安装在系统中比较方便。而有些另外的模块并不是每个项目都需要，所以只是在特定的项目中才会选择去安装这种模块。
